## 快速开始

```shell
# clone 项目代码至本地
git clone https://github.com/hanyuancheung/gpt-go.git

# 进入项目目录
cd gpt-go

# 设置API_KEY为环境变量
export API_KEY={YOUR_API_KEY} chatgpt

# go build example 二进制
make chatgpt-example

# 运行 example
./chatgpt
```

- [x] 对 Completion API 的流式支持
- [x] 文档搜索 API
- [x] 图片生成 API
- [x] 替换默认 url、用户代理、超时和其他选项
